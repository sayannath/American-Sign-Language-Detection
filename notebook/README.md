# American Sign Laguage Detection

### Dataset Used

## Architecture

[MobileNetV2](https://arxiv.org/pdf/1801.04381.pdf) [Used this Architecture to deploy it in Android]

[MobileNetV3Large](https://arxiv.org/pdf/2008.08178.pdf)

## Model Size - [.h5]

> MobileNetV3Large - 24MB
